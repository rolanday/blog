
This project references the following libraries:

Newtonsoft.Json
protobuf.net

The can be added using NuGet (search for Json.Net and protobuf-net,
respectively) or manually by downloading the the requires assemblies
from the following sites:

http://json.codeplex.com/releases/view/97986
https://code.google.com/p/protobuf-net/
